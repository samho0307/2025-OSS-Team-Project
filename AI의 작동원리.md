# 데이터 수집 (Data Collection)

AI는 학습을 위해 매우 많은 양의 데이터를 필요로 한다. 이 데이터는 텍스트, 이미지, 소리, 영상, 숫자 등 다양한 형태일 수 있다. 예를 들어 음성 인식 AI라면 수천 시간 분량의 사람 목소리를, 이미지 인식 AI라면 수많은 사진과 그에 대한 설명이 필요하다.

즉, 데이터의 양과 질이 AI의 성능을 크게 좌우한다.

# 데이터 처리 및 정제 (Preprocessing)

수집한 데이터에는 불필요한 정보, 오류, 중복된 내용이 포함되어 있을 수 있다. 따라서 AI가 효율적으로 학습할 수 있도록 데이터를 정리하고 불필요한 부분을 제거하는 과정이 필요하다.

이 단계에서는 데이터 정규화, 노이즈 제거, 분류, 라벨링 등의 작업이 이루어진다.

# 학습 (Training)

학습 과정에서는 AI가 데이터를 분석하여 패턴과 규칙을 스스로 찾아내는 과정이 이루어진다. 이때 주로 사용되는 기술이 머신러닝과 딥러닝이다.

머신러닝: 데이터 속 패턴을 스스로 학습

딥러닝: 여러 층으로 이루어진 인공신경망을 통해 더 복잡한 패턴을 학습

이처럼 AI는 반복적인 계산과 비교를 통해 점점 더 정확한 결과를 낼 수 있도록 스스로를 개선한다.

# 모델 생성 (Model Building)

학습이 완료되면 AI는 하나의 **모델(Model)**을 만든다. 이 모델은 학습 결과가 저장된 ‘두뇌’와 같으며, 이후 새로운 데이터를 입력받으면 그에 대한 결과를 예측하거나 판단할 수 있다.

좋은 모델을 만들기 위해서는 다양한 데이터를 사용하여 충분히 학습하는 것이 중요하다.

# 예측 및 실행 (Inference)

완성된 모델은 실제 상황에서 사용된다. 예를 들어 새로운 사진을 입력하면 ‘이것은 고양이다/개다’를 판단하거나, 문장을 입력하면 번역해 주거나 답변을 생성한다.

이 과정을 **추론(Inference)**이라고 하며, 우리가 AI를 사용할 때 경험하는 대부분의 기능이 이 단계에 해당한다.
